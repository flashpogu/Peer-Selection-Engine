# 🤖 AI-Assisted Peer Selection & Swarm Scheduling Engine

A distributed peer-to-peer (P2P) file transfer system that uses machine learning to intelligently select peers and optimize swarm scheduling.

---

## 🚀 Project Overview

This project simulates a **torrent-like P2P system** where multiple peers exchange file chunks.  
Unlike traditional systems, peer selection is **AI-driven**, improving efficiency and adaptability.

---

## 🧠 Key Features

- 🤖 **AI-Based Peer Selection**
  - Uses a trained ML model to choose the best peer
  - Considers metrics like upload rate, reliability, latency

- 🔄 **Swarm Scheduling Engine**
  - Dynamic per-chunk peer selection
  - Adaptive download strategy

- 🌐 **Distributed System (LAN Ready)**
  - Supports multiple devices on same network
  - Central tracker for peer discovery

- 📦 **Chunk-Based File Transfer**
  - Files split into chunks
  - Parallel-ready architecture

- 📊 **Real-Time Dashboard**
  - Live peer monitoring
  - Download progress visualization
  - AI decision tracking
